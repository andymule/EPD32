Wiring:

ESP32 / Waveshare 2.9" black white e-paper

- VCC / 3V
- GND / GND
- DIN / 23
- CLK / 18
- CS / 5
- D/C / 17  
- RST / 16
- BUSY / 4

WIRING UPDATE JAN 2020:
  using Lilygo T5 v2.2, no external wiring needed
